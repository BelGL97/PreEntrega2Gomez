IconifyIcon web component makes using icons as easy as icon fonts. To use "facebook" in HTML, add this code to the document:

<iconify-icon icon="mdi:facebook" width="24" height="24"></iconify-icon>

Web component will load icon data from Iconify API and render SVG.

Make sure you import IconifyIcon web component:

<script src="https://code.iconify.design/iconify-icon/1.0.2/iconify-icon.min.js"></script>

Web component works with almost all modern UI frameworks. Click documentation link below.
